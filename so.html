Processo de Instalação Passo a Passo
01

Criar USB Bootável

Descarrega a ISO oficial do Windows 11 em microsoft.com. Usa o Rufus (Windows) ou Balena Etcher (multiplataforma) para criar um USB bootável de pelo menos 8GB. Seleciona GPT + UEFI no Rufus para hardware moderno.

02

Configurar a BIOS/UEFI

Entra na BIOS (DEL / F2 no arranque). Ativa Secure Boot e confirma que o TPM 2.0 está ativo. Define o USB como primeiro dispositivo de arranque (Boot Order). Guarda e reinicia.

03

Arrancar a partir do USB

Arranca a partir do USB de instalação. Escolhe idioma, hora e teclado. Clica em "Instalar Agora". Introduz a chave de produto ou seleciona "Não tenho chave de produto" para ativar depois.

04

Particionamento do Disco

Escolhe "Instalação Personalizada". Apaga as partições existentes no disco destino ou cria uma nova. O instalador cria automaticamente as partições necessárias (EFI, MSR, Windows). Para dual-boot, reserva espaço não alocado antes de instalar.

05

Instalação e Configuração Inicial

O Windows copia e instala os ficheiros (~15-30 minutos). Após o reinício, o OOBE guia a configuração: região, teclado, conta Microsoft (ou local), privacidade, OneDrive. Cria o teu utilizador e palavra-passe.

06

Drivers e Atualizações Pós-Instalação

Windows Update instala drivers básicos automaticamente. Para performance máxima: instala drivers da GPU (NVIDIA/AMD do site oficial), chipset da motherboard, e drivers de áudio e rede. Executa o Windows Update até não haver mais atualizações. Instala o software essencial (browser, antivírus, etc.).
